REACT'S PURPOSE

React is a Javascript Library for building composable user interfaces. It encourages the creation of reusable UI components which present data that changes over time. It shares simple declarative power with HTML interfaces that create dynamic data. It was created for flexibility and reusability, it is easy to use and fast.
